MapUp

An app to enhance the search experience of the Meetup app by adding an interactive map to provide the user with a visual representation of the location of technology Meetups. It provides a map interface to find Meetups based on location. This app utilizes: Google Maps API, Geocode API, and Meetups API Languages: JavaScript, jQuery, AJAX, SQL (storage of API data), HTML, CSS

Resume Hackers

Desktop app for creating a personally customized resume that is perfectly formatted. The user adds their information and the resume is pre-formatted using an elegant template. The resume dynamically updates as information is added to it. Languages: JavaScript,jQuery, HTML, CSS
